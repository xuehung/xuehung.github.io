Smart Home Box
There are many sensors installed at the future smart home environment. In order to provide a platform for sensors to work together and assist users install devices easily, I implemented a gateway called Smart Home Box by imploying Apache ActiveMQ as the communication protocol and JSON as the format for data exchange. There are two core functions: device discovery and service composition. Device discovery enables the gateway to react to addition and removal of sensors. It automatically fetches the newly added sensor's manifest file which describes its capabilities so that users can operate sensors without the prior knowledge. Service composition is responsible for finding an alternative device when a sensor failed. To do so, we can prevent the running service from being interrupted.
